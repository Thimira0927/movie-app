# 🎬 Movie Search Web App

A simple and responsive web application that allows users to search for movies and view detailed information.

## 🔗 Live Demo
https://thimira0927.github.io/movie-app/

## 🚀 Features
- Search for movies
- View movie details (title, year, genre, cast)
- IMDb rating display
- Watch trailer option
- Clean and modern UI

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
